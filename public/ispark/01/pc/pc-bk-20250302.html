구조, 열, 전자, 유동
등의 해석 기능을 갖춘 SIMULIA

SIMULIA는 다쏘 시스템(Dassault Systèmes)이 제공하는
첨단 엔지니어링 시뮬레이션 소프트웨어로, 제품 설계, 성능 예측, 최적화 등을 지원합니다.
시뮬리아는 구조 역학, 열역학, 유체역학 및 전자기 현상과 같은 다양한 분야를 포괄하며,
제품 개발 초기 단계에서 잠재적 문제를 미리 발견하여 더 나은 의사결정을 내릴 수 있도록 돕습니다.

이즈파크는 SIMULIA의 고급 해석 도구를 통해 고객의 요구에 맞는 최적의 엔지니어링 솔루션을 제공하여
설계 초기부터 최적의 성능을 보장하고, 효율적인 문제 해결과 혁신적인 결과를 이끌어냅니다.

구조/피로

다양한 산업 분야에서 복잡한 기계적 구조의 거동을 예측하고,
최적화된 설계와 성능을 보장하기 위해 필수적인 해석 기법을 제공합니다.

선형해석 (Linear Analysis)

  • 적용 분야:
    빌딩, 항공기, 자동차 부품의 초기 설계 검증
  • 분석 대상:
    미소 변형, 선형 하중 조건

피로/내구도 해석 (Fatigue/Durability Analysis)

  • 적용 분야:
    자동차 부품, 철도 차량, 건축 구조물, 항공기 부품의 피로 수명 예측
  • 분석 대상:
    반복 하중에 의한 손상 누적, 제품수명

파괴해석 (Fracture Analysis)

  • 적용 분야:
    항공기 날개, 터빈 블레이드, 고속 기계 부품의 파괴
  • 분석 대상:
    균열 발생 및 성장, 파손 기전

비선형해석 (Nonlinear Analysis)

  • 적용 분야:
    충돌 분석, 고온 환경에서의 재료 거동, 금속 성형, 복합재 구조 해석
  • 분석 대상:
    대변형, 접촉 문제, 재료 비선형성
전자기장/ EM

전기 및 자기장의 분포와 상호작용을 정확하게 시뮬레이션하여,
다양한 전기 시스템, 통신 장비, 전자기기의 성능을 최적화할 수 있도록 지원합니다.

전자기장해석 (Electromagnetic Field Analysis)

  • 적용 분야:
    전자기기, 안테나, 인쇄기판, 반도체 패키지, 우주항공, 인공위성, 전력시스템
  • 분석 대상:
    전자기장 분포, 전자기파 간섭, 신호 무결성, 전력 무결성, 방사패턴

고주파대역 해석 (Radio Frequency Analysis)

  • 적용 분야:
    무선 통신, 레이더 시스템, 안테나 설계, 위성 통신, 필터, 커넥터
  • 분석 대상:
    고주파 전파, 신호 반사 및 흡수, 전파 손실, 신호 간섭, 노이즈 분석, SAR, ESD

저주파대역 해석 (Low Frequency Analysis)

  • 적용 분야:
    변압기 설계, 전기 모터 최적화, 센서, 전력 시스템 안정성 분석
  • 분석 대상:
    로렌츠힘, Electrostatic Chuck, 저주파 전자기 유도, 정전기, 유도기 전력(EMF)
열

제품과 시스템의 열 성능을 정확하게 분석하여 안정성, 내구성, 효율성을 극대화하는 데 필요한 포괄적인 해석 기능을 제공합니다.

열해석 (Thermal Analysis)

  • 적용 분야:
    전자기기 냉각 시스템, 자동차 엔진 냉각, 자동차 부품 열 변형
  • 분석 대상:
    열전도, 대류, 복사, 열방사

전기-열해석 (Electro-Thermal Analysis)

  • 적용 분야:
    전자기기 냉각, 전기차 배터리 관리, 고전력 전자 부품
  • 분석 대상:
    줄열(JouleHeating), 열폭주, 전기저항, 재료의 전기-열적 특성

열-유동해석 (Thermal-Fluid Flow Analysis)

  • 적용 분야:
    엔진 냉각 시스템, 전자기기 열 관리, 열교환기 설계 최적화
  • 분석 대상:
    자연대류, 강제대류, HVAC
최적화 및 강건설계

제품 설계에서 불필요한 요소를 제거하고 강도, 무게, 효율성을 동시에 개선하여 제품의 성능을 극대화하고, 비용 효율성을 높일 수 있습니다.

형상최적화 (Shape Optimization)

  • 적용 분야:
    항공기 날개 설계, 자동차 부품, 고층 건축 구조 설계
  • 분석 대상:
    제품 경량화, 응력 분포, 강도 최적화, 모핑(Morphing)

변수최적화 (Parameter Optimization)

  • 적용 분야:
    엔진 성능 최적화, 전자기기 냉각 시스템 설계, 기계 부품의 피로 수명 연장
  • 분석 대상:
    AI, 실험계획법(DOE), 최적화 분석, 6시그마, 파레토 분석, 메타 / 써로게이트 모델
진동/소음/NVH

제품의 진동특성 및 소음 특성을 정밀하게 분석하여 NVH 성능 향상에 기여하는 포괄적인 해석 기법을 제공합니다.

진동해석 (Vibration Analysis)

  • 적용 분야:
    기계 장비의 진동 저감 설계, 항공기 진동 안정성 분석, 고속 열차의 구조적 진동 분석
  • 분석 대상:
    NaturalFrequency, Steady-State Dynamics, Harmonics, Random, Eigen-mode

음향해석 (Acoustic Analysis)

  • 적용 분야:
    차량 소음 분석, 항공기 소음 저감, 건물의 음향 설계
  • 분석 대상:
    Sound Pressure Level(dB), Particle Acceleration, Noise Criteria(NC)
유체역학 및 유동

다양한 산업 분야에서 복잡한 유체 흐름을 정확하게 분석하고, 최적의 설계를 지원하기 위해 필수적인 해석 기법을 제공합니다.

유동해석 (Fluid Flow Analysis)

  • 적용 분야:
    항공기 공기역학 분석, 자동차의 공기역학 설계, HVAC 시스템 최적화
  • 분석 대상:
    유속, 유량, 난류, 항력, 양력, Pressure Drop, 프로펠러

유체-구조 상호작용 해석 (Fluid-Structure Interaction, FSI)

  • 적용 분야:
    항공기 날개의 변형 해석, 해양 구조물 설계, 교량의 바람 하중 해석
  • 분석 대상:
    유체 하중, 구조물 변형, 유속, 유량
다물체동역학

복잡한 기계 시스템의 동적 거동과 진동 특성을 정밀하게 분석하여 설계 최적화와 성능 향상에 기여하는 포괄적인 해석 기법을 제공합니다.

다물체동역학 (Multibody Dynamics)

  • 적용 분야:
    자동차 엔진 및 파워트레인, 기계 시스템, 산업용 로봇, 철도 레일
  • 분석 대상:
    부품 간 동적 상호작용, 거동 메커니즘

진동해석 (Vibration Analysis)

  • 적용 분야:
    기계 장비의 진동 저감 설계, 항공기 진동 안정성 분석, 고속 열차의 구조적 진동 분석
  • 분석 대상:
    고유 진동수, 모드 형상, 진동 저감
공정

다양한 재료의 물리적 특성과 제조 과정에서 발생할 수 있는 문제를 정확하게 예측하고, 최적의 제조 솔루션을 제공합니다.

사출해석 (Injection Analysis)

  • 적용 분야:
    자동차 플라스틱 부품, 전자기기 외장재, 의료기기, 소비재 제품
  • 분석 대상:
    재료 흐름, 충진 / 보압/ 냉각 시간, 수축 및 변형

단조해석 (Forging Analysis)

  • 적용 분야:
    조선 부품, 자동차 샤시 부품, 중공업 기계 부품, 금속 구조물
  • 분석 대상:
    고온 / 온열 / 냉간 단조, 스프링백 현상, 아이어닝
